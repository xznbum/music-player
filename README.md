music-player
============

Проблема: отсутствие общедоступного плеера/хранилища музыки.


Цель проекта: разработать онлайн аудио-проигрыватель с возможностью поиска музыкальной композиции по названию/исполнителю/жанру/, добавления (загрузки) собственных аудио-файлов, создания личных списков воспроизведения (плейлистов), просмотра наиболее популярных (прослушиваемых) композиций. Проект должен быть завершен в 2014 году.

+возможность голосования и комментирования композиций, репост музыкальной композиции в соц. сеть ВКонтакте.
